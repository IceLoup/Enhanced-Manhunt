## Enchanced Manhunt Plugin

### Many Updates are coming soon such as twists !

This is an _easy to use_ Manhunt plugin including many features
Play you best manhunt games with this plugin!

- Multi Runners support
- Nether and others dimension tracking support
- Friendly Fire managment
- Head Start managment
- Timer
- WorldBorder managment
- Compass tracking managment
- Win event
- Spectator option and support

Full GUI configuration is possible via 
```
/manhunt
```

Full Ingame Configuration


Try it out now on modrinth https://modrinth.com/plugin/enchanced-manhunt
